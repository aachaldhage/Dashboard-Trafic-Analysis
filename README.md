# 🚦 Traffic Analysis Dashboard

## 📌 Project Overview

The **Traffic Analysis Dashboard** is a Power BI project designed to analyze and visualize traffic data. It helps in understanding traffic patterns, congestion levels, accident trends, and vehicle distribution using interactive visualizations.

---

## 🎯 Objectives

* To analyze traffic flow across different locations
* To identify peak traffic hours
* To study accident patterns and causes
* To create an interactive dashboard using Power BI
* To support better traffic management decisions

---

## 📊 Dataset Description

This project uses two datasets:

1. **Traffic Data**

   * Traffic ID
   * Date
   * Location
   * Vehicle Count
   * Traffic Density
   * Average Speed

2. **Accident Data**

   * Traffic ID (Common Column)
   * Number of Accidents
   * Accident Severity
   * Weather Condition
   * Time of Accident

---

## 🔗 Data Relationship

* The common column **Traffic ID** is used to create a relationship between both datasets in Power BI.

---

## 📈 Visualizations Used

* 📊 **Stacked Column Chart** – Traffic volume over time
* 📉 **Line Chart** – Traffic trends
* 📊 **Bar Chart** – Location-wise traffic comparison
* 🥧 **Pie Chart / Donut Chart** – Vehicle distribution
* 🔢 **Cards** – Total vehicles, total accidents
* 🎛️ **Slicers** – Date, location, weather filters
* 🗺️ **Map (optional)** – Traffic by location

---

## ⚙️ Tools & Technologies

* Microsoft Power BI
* Excel / CSV datasets
* Data Cleaning & Transformation (Power Query)

---

## 🧹 Data Cleaning Steps

* Removed null values
* Standardized column names
* Converted data types
* Created relationships between tables

---

 Key Insights

* Peak traffic occurs during morning and evening hours
* High accident rates are observed during bad weather
* Certain locations have consistently high traffic density
* Traffic speed decreases as vehicle count increases

---

Conclusion

This dashboard provides meaningful insights into traffic behavior and helps in making data-driven decisions for traffic management and planning.
